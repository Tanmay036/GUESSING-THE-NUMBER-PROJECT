# GUESSING-THE-NUMBER-PROJECT
 The Number Guessing Game is a simple console-based game developed in the C programming  language. The game randomly selects a number within a given range, and the player tries to guess it.  After each guess, the program provides feedback‚Äîindicating whether the guessed number is too high,  too low, or correct.

 1. This project demonstrates the use of basic programming concepts such as:
 - Random number generation
 - Loops and conditional statements
 - User input/output
 - Logic implementation and control structures
 The main goal is to improve understanding of fundamental C programming and logical problem-solving 
skills.

---

 2. Objectives
 The primary objectives of the Number Guessing Game project are:
- To develop a simple and interactive console-based game using C language.
- To understand and implement loops, conditional statements, and functions in C.
- To demonstrate random number generation using the standard C library.
- To enhance problem-solving and debugging abilities
